Some of my projects

Natours

Natours is a tour company's website, which shows tour plans and their offers in a simple and elegant design. I designed this site in a fully responisve manner, in the form section I have designed a custom radio button. Designed with HTML and SASS preprocessor .

Forkify

Search over 100k recipes with forkify, this is a recipe finder web app you can instanly add ingredients to the shopping cart. Add your favourite recipes to the favourites list, it persist the data in the browser. Designed using HTML, SASS, JS,webpack and Forkify API

NewsGrid

NewsGrid is a news providing website, which offers instant news. This is not a fully functional website yet. It is a fully responisve site and used CSS-Grid for layouts, which is interesting a challenging task for me!. Designed using HTML and CSS-Grid

Tracalorie

Tracalorie is a CRUD application which tracks your calories. You can create, update, and delete your calorie records any time. It persist the data in the browser. Designed using Bootstrap and JS

Github Finder

GitHub finder is a site to search a GitHub user. Fetches the github users profile and their recent public repositories, public gists and followers count. Designed using Bootstrap, JS and GitHub users API

PostBook

PostBook is CRUD application to share your thoughts, It is a prototyping application, as for now runs with postbook API. Designed using HTML CSS and JS

Dice Game

A simple dice game for kids. You can set the max winning points, and hold your dice numbers a your score. Designed and developed using HTML CSS and JS

Student Details

Student Details is a full-stack web application to store and search student's details. Which has a login system for authentication, firebase auth provides authentication servies and firestore for data. Designed and Developed using HTML, CSS, JS and FIREBASE

808 Drum pad
